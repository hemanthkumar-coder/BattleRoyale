# 16-Week Milestone Plan

## Weeks 1–2: Preproduction
- Lock GDD MVP.
- Finalize art/tech constraints.
- Choose engine and backend stack.

## Weeks 3–4: Core Controller + Combat
- Movement and camera complete.
- 3 starter weapons.
- Damage, hit markers, death state.

## Weeks 5–6: Multiplayer Foundation
- Dedicated server flow.
- Match join/leave and reconnect.
- Replicated health, ammo, elimination.

## Weeks 7–8: Battle Royale Loop
- Drop phase.
- Loot generation and pickups.
- Safe zone phase controller.

## Weeks 9–10: Team Systems + UX
- Duo team logic and revive.
- HUD, minimap, kill feed.
- Spectator mode.

## Weeks 11–12: Backend + Progression
- Account auth and inventory profile.
- Match result persistence.
- XP and level progression.

## Weeks 13–14: Optimization + Scale
- CPU/network profiling.
- AOI/replication tuning.
- Raise concurrent players.

## Weeks 15–16: Beta Readiness
- Bug bash and exploit hardening.
- Balance passes based on telemetry.
- Release candidate build and ops playbook.
